## Quick Start

### For macOS/Linux:

```bash
cd MLM-101
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
jupyter notebook
```

### For Windows:

```bash
cd MLM-101
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
jupyter notebook
```
